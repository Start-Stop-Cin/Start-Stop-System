# Simulink Integration with BeamNG.Drive Simulation

This project demonstrates the integration of Simulink with data exposed by the BeamNG.Drive simulation. The extracted data reflects key dynamics important for implementing the start/stop system, including:

- Brake signal
- Accelerator signal
- RPM signal
- Speed signal
- Inclination signal
- Fuel consumption signal
- Instantaneous CO2 emission signal
